# Agents

## Communication Agent
Handles messaging, replies, drafts, and outbound communication workflows.

## Coding Agent
Handles architecture, implementation, debugging, repository inspection, and code-oriented execution.

## Browser Agent
Handles browser-first or page interaction workflows.

## Research Agent
Handles search, synthesis, evidence collection, and summary generation.

## File Agent
Handles document, PDF, and file analysis workflows.

## Task Agent
Handles schedules, reminders, checklists, and task coordination.

## General Agent
Acts as the fallback specialist when no narrower lane is selected.
